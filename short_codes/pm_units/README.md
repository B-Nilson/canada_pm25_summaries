# pm_units Extension For Quarto

Shorthand code for HTML units of PM2.5 concentration ("micro grams per metre cubed")

## Installing

```bash
quarto add B-Nilson/pm_units
```

This will install the extension under the `_extensions` subdirectory.
If you're using version control, you will want to check in this directory.
